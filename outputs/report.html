一、主题选择

AI 编程工具与 Agent：从代码补全到自主编码的演进，以及其中的个人机会

选题依据：编程工具是学习者每天都在使用的东西，信息差直接转化为效率差和成本差；2026 年上半年这个赛道发生了计费模式转型、标准中立化等结构性变化，信息雷达的价值最大化。

聚焦范围：编程 Agent 产品、开源编码 Agent、国产方案、基准测试、计费与成本、MCP/AGENTS.md 标准、相关比赛机会。

    • 三、核心信息分析（四条主线）

    主线一：编程 Agent 能力演进——"补全工具"已死

    六大平台（Claude Code、Cursor、Copilot、Codex、Antigravity、Devin Desktop）已全部具备自主编码闭环。SWE-bench 头部模型集中在 70-77%，第 1 名与第 8 名仅差约 6 分——模型能力差距收敛，竞争的焦点从模型转向工作流（harness）。

    主线二：计费剧变——Token 成为新的成本单位

    GitHub Copilot 2026-06-01 起全面转向 AI Credits 按量计费，全员跟进。DX 研究显示中位 PR 吞吐量增益仅 7.76%，远低于厂商宣称的 3 倍。实际人均成本 $200-600/月。个人应对：国产低价 API（DeepSeek 约 ¥2/百万 Token）配合开源 Agent 可大幅压缩成本。

    主线三：开源与国产方案——个人门槛坍塌

    中国市场 2025 年 3.99 亿 → 2026 预计 11.73 亿元。通义灵码升级为 Qoder CN 矩阵，CLI 以 Apache 2.0 开源。国产模型（DeepSeek、Kimi、MiniMax）在 SWE-bench 上进入 70+ 区间，与国际一线的工程差距缩小到可接受范围。

    主线四：标准化——MCP 与 AGENTS.md 定义新基础设施

    MCP 于 2025-12 捐入 Linux 基金会 AAIF，SDK 月下载约 1 亿次，MCP 1.0 于 2026-06-24 发布。AGENTS.md 经实证确认为跨工具配置 Agent 行为的事实标准。个人机会：围绕标准协议做差异化（MCP Server 开发、Agent 配置工程）。

    四、价值判断

    层级内容 强信号按量计费转型、MCP 中立化、模型分差收敛、PR 增益仅 7.76%、AGENTS.md 标准化 中等信号国产模型排名（需持续验证）、市场规模预测（方向可信、数字不必当真） 噪音极端账单个案、"淘汰程序员"叙事、厂商发布会效率宣称

    一个关键的交叉验证实例：同一基准（SWE-bench Verified）出现了两个"都对"的分数（88.6% vs 76.8%），差异来自评测口径（厂商最优 harness vs 官方统一 harness），比较数据前必须先对齐测量方法。

    五、风险边界

  • 时效性风险：价格信息半衰期以季度为单位，基准分数每月漂移
  • 来源偏差：厂商分数普遍使用最优配置，需第三方复测验证
  • 网络与合规：部分国际工具在国内直连不稳定
  • 学习误区：追新工具 ≠ 学习；AI 写得出来 ≠ 你会了；信息收藏 ≠ 信息优势
  • 报告边界：只覆盖中英文公开网络，未覆盖付费报告和闭门社区

    七、任务复盘

    方法总结

    1. 定主题（"与我相关 × 变化够快 × 信息差够大"三条标准） 2. 设检索矩阵（6 个维度 = 报告 4 条主线 + 补充角度 + 比赛机会） 3. 记录来源（每条标注链接、时间、一手/二手属性，杜绝无来源引用） 4. 过滤分层（五条标准 = 一手优先 / 结构性 > 事件性 / 有数字有方法 / 反直觉 / 可行动） 5. 行动转化（每条建议标注依据编号，信息 → 判断 → 行动可回溯）

    信息雷达持续运转机制

    频率动作 每周 30 分钟快扫固定信源，只记结构性变化 每月 1 次深读当月最重要变化 + 一次最小实验 每季 1 次复盘上季度判断准确率，校准过滤标准

    最小实验补充

    已完成一次真实 bug 修复实验（deepseek-v4-pro API）：4/4 测试通过，单次 631 tokens，成本 < ¥0.01。实验结果附带发现：Windows 中文 JSON 编码坑、推理模型响应结构差异（thinking+text 双块）。完整过程文件保留在 `experiment/` 目录，可独立复现。

    一句话结论

    信息壁垒的本质不是"知道得多"，而是固定的信源、明确的过滤标准、强制的行动转化三者形成闭环。
